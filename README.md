# MERN Blog App — Toronto Blogs

A full-stack blog application built from scratch while learning the MERN stack. This is my first complete MERN project (1/2).

## What I Learned
- Designing databases from entity identification to Mongoose schemas
- Building RESTful APIs with Node.js and Express
- JWT authentication with secure httpOnly cookies
- Role-based access control (Admin vs User)
- Password hashing with bcrypt
- Centralized API services with Axios
- Global error handling with Axios interceptors
- State management with Redux Toolkit
- Protected routes in React
- Auth persistence across page refreshes

## Tech Stack
- **MongoDB** - Database
- **Express.js** - Backend framework
- **React.js** + Vite - Frontend
- **Node.js** - Runtime environment
- **Redux Toolkit** - State management
- **Tailwind CSS v4** - Styling
- **Axios** - HTTP client
- **react-hot-toast** - Notifications
- **bcryptjs** - Password hashing
- **jsonwebtoken** - JWT authentication

## Features
- User registration and login
- JWT authentication with httpOnly cookies
- Auth persistence across page refreshes
- Admin dashboard to manage posts
- Admin can create, update, and delete posts
- Category management (create and delete)
- Users can read posts and leave comments
- Vintage warm design theme
- Protected routes for admin pages
- Role-based UI rendering
