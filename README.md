# MERN Blog App

A full-stack blog application built while learning the MERN stack from scratch.

## What I'm Learning
- Designing databases from entity identification to Mongoose schemas
- Building RESTful APIs with Node.js and Express
- JWT authentication with secure httpOnly cookies
- Role-based access control (Admin vs User)
- Password hashing with bcrypt
- Connecting frontend and backend with Axios
- State management with Redux Toolkit
- Protected routes in React

## Tech Stack
- **MongoDB** - Database
- **Express.js** - Backend framework
- **React.js** - Frontend
- **Node.js** - Runtime environment

## Features
- User registration and login
- JWT authentication with cookies
- Admin can create, update, delete posts
- Users can read posts and comment
- Category management
